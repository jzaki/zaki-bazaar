# Zaki Bazaar — The finest (and coarsest) modules from across the lands

Modules I make, like, or otherwise think are worth sharing.

## Travelling to the bazaar...

1. Get the latest Basecamp 0.2.0 - https://github.com/logos-co/logos-basecamp/releases
(download the relevant binary for your system)

2. Open basecamp and click the gear icon (Settings), then **Package Repositories**.
3. Add Zaki Bazaar url -> https://raw.githubusercontent.com/jzaki/zaki-bazaar/refs/heads/main/logos-repo.json

4. Now go to Applications, and under Zaki Bazaar you'll see the apps I've added, namely [EX] for **Example Forum** (only shows "Example..." in the icon text).
5. Install it (and the logos-delivery dependency), then load the app by clicking on it.
